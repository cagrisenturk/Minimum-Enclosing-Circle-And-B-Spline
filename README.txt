AFTER THAT OPEN THE C FILE, GO > PROJECT SETTINGS > LINKERS AND WRITE
-lbgi
-lgdi32
-luser32
AND LAST RUN THE CODE!

{-12,6}
{8,-5}
{-3,-7}
{3,6}
{14,1}
